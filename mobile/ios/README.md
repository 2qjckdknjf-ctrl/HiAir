# HiAir iOS

## Generate project (XcodeGen)

1. Install XcodeGen:

```bash
brew install xcodegen
```

2. Generate Xcode project:

```bash
cd /Users/alex/Projects/HIAir/mobile/ios
xcodegen
```

3. Open and run:

```bash
open HiAir.xcodeproj
```

The generated app starts from `HiAirApp` and uses `RootTabView`.
