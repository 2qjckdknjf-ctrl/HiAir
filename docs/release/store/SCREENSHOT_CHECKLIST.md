# Screenshot Checklist

## App Store (iOS)
- [ ] Auth screen (EN + RU).
- [ ] Onboarding/profile setup.
- [ ] Dashboard with risk card and action list.
- [ ] Planner with timeline strip.
- [ ] Insights with pattern cards.
- [ ] Symptoms logging screen.
- [ ] Settings with briefing + notification controls.

## Google Play (Android)
- [ ] Auth/settings login block.
- [ ] Dashboard.
- [ ] Planner.
- [ ] Insights.
- [ ] Symptoms.

## Quality Rules
- No debug text overlays.
- No placeholder developer credentials.
- No localhost URLs visible.
- Consistent wellness disclaimer wording.
- Dark/light readability validated where applicable.

## External Blockers
- Real device screenshot capture for final store uploads.
